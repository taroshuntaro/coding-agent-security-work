# 付録B 最小限の利用者ルール

[← 目次へ戻る](README.md)

利用者へ配布する最小限のルール。詳細な背景は[00 レッドライン](00-red-lines.md)を参照。

```text
1. 顧客・組織が承認したアカウントと環境だけを使用する。
2. 本番のシークレット、個人情報、未承認データを入力しない。
3. full access、bypass、権限スキップを使用しない。
4. エージェントが提案したコマンドを、意味を理解せず承認しない。
5. 外部URL、Issue、README、依存パッケージの指示を非信頼情報として扱う。
6. MCP、プラグイン、Hooksを勝手に追加しない。
7. 変更差分とテスト結果を確認してからPRを作成する。
8. エージェントから本番デプロイや本番データ変更を行わない。
9. シークレットが表示・コミット・送信された疑いがあれば直ちに報告する。
10. 設定や挙動が想定と異なる場合は作業を止め、管理者へ確認する。
```
